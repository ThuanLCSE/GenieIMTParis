Comment ça marche

L'acheteur

  1. Crées une liste de courses avec le magasin souhaité et laisse ton numéro pour le texto, ou bien inscris-toi afin de profiter de la fonction tchat.
  2. Attends qu'un Voisin t'envoie un texto ou un tchat (tu recevras une notification).
  3. Négocies avec lui les frais de livraison (c'est possible de mettre 0 EUR, mais une participation rendra les deux heureux):
  4. Récupère tes courses lorsque le Voisin les livre et mets le code de vérification sur l'écran du Voisin (le code est affiché dans ton écran de tchat).

Le Voisin

  1. Connectes-toi sur Bonjour Courses.
  2. Quand tu trouves une courses qui te vas, au niveau du trajet et du magasin, soit tu envoies un texto perso au numéro de téléphone, soit tu t'inscris pour profiter de la fonction tchat.
  3. Quand tu es dans de tchat, cliques sur l'option "je t'aide" dans le menu "Action" pour accepter la demande.
  4. D'abord, lis avec attention les consignes de courses, en particulier les réglementations des surgelés. Ensuite tu paies les courses. Tu peux demander à l'acheteur de te faire un virement et demander la preuve du paiement pour être sûr.
  5. Quand tu livres les courses à domicile, cliques sur l'option "Courses livrés" dans le menu "Action" pour valider la livraison. Pour cela, tu as besoin du code de vérification de l'acheteur.
